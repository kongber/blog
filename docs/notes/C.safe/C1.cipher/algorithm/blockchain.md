## 1. 区块链

### 1.1 基本概念

- 区块链是一种分布式账本技术，由多个节点组成，每个节点都有一份完整的账本副本。
- 账本由区块组成，每个区块包含了一定数量的交易记录。
- 每个区块都有一个唯一的哈希值，哈希值由区块的前一个哈希值、交易记录和时间戳等因素计算得出。
- 每个节点都可以验证其他节点的交易记录是否合法，通过验证哈希值来判断。
- 因为每个节点都有一份完整的账本副本，所以区块链是去中心化的，没有中心化的 authority。
